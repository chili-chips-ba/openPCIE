# WIP -- Under Construction: Opensource RC-switched core (bonus)

This is a yet to be done opensource variant of what was first tested with [AMD-proprietary RC-switched IP](../../2.amd-rtl-with-Vivado-build/3.Bonus--RC-switched.amd) stack. This would be an extra / bonus activity, above and beyond our original plans.

The objectives of this additional dev track are to:
 - first design an opensource _RC-switched_ core, based on the already tested and familiarized with proprietary core
 - then validate and showcase its operation in a Switched PCIE context, on the real hardware, repeating the test procedure used for AMD solution.

With both this indirect/switched and [RC-direct](../2.RC-direct.opensource) use-cases thus tried and proven to be true, we will have established a nicely rounded set of opensource cores and examples -- A solid foundation for the makers to build their future applications upon...
