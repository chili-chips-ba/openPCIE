# Opensource PCIE End-Point for AMD Series7 FPGAs

This is the opensource counterpart of AMD-proprietary EP core. It is used only for testing of our Root Complex core, and provided solely for completness -- You are free to build any one of them and try its operability with this Root Complex. They should all work equally well.

For the replica of the AMD EP please go to [../../2.amd-rtl-with-Vivado-build/1.EP.amd](../../2.amd-rtl-with-Vivado-build/1.EP.amd).

We have not replicated the opensource End Point designs here. They can be found in their respective repos, built on the side, then used in a test setup with the Root Complex that this project is about.

1) [Basic PCIE EP for LiteFury](https://github.com/hdlguy/litefury_pcie)
2) [Regymm PCIE](https://github.com/regymm/pcie_7x)
3) [LiteX PCIE EP](https://github.com/enjoy-digital/litepcie)
